How To Use This Page

This page gathers the project's open questions in one place. Each question points back to the page where that thread is explored, so readers can move from question to research, then back to the wider map.

Center of gravity: The questions wind around one recovery-oriented theme: how psilocybin and related compounds might support mood, behavior change, neuroplasticity, learning, and integration while keeping evidence categories clear.
